# AGENTS Instructions

## Startup

Read `core/agent_rules/foundation_startup.md` and `core/standards/governance_structure_standard.md` before changing shared governance. Canonical governance documents are written in English. Apply the core-first placement test before adding a platform or profile contract.

## Repository Boundary

This repository contains shared rules, skills, and workflows. Do not add game-specific paths, domain names, runtime content, project snapshots, or active product plans. Core must remain engine- and framework-neutral; irreducible technology contracts belong below the selected platform.
